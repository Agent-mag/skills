# find-skills

> Helps users discover and install agent skills when they ask questions like "how do I do X", "find a skill for X", "is there a skill that can...", or express interest in extending capabilities. This skill should be used when the user is looking for functionality that might exist as an installable skill.

**Category:** Memory & Knowledge · **License:** MIT

## Install

```bash
npx agentmag add skill find-skills
```

## What This Skill Does

This skill provides a reusable prompt configuration for AI coding agents. When installed, it adds structured instructions that guide your agent's behavior for **find-skills** tasks.

## Compatibility

- Claude Code
- Cursor
- Windsurf
- Copilot
