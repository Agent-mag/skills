# write-a-skill

> Create new agent skills with proper structure, progressive disclosure, and bundled resources. Use when user wants to create, write, or build a new skill.

**Category:** Code & Compute · **License:** MIT

## Install

```bash
npx agentmag add skill write-a-skill
```

## What This Skill Does

This skill provides a reusable prompt configuration for AI coding agents. When installed, it adds structured instructions that guide your agent's behavior for **write-a-skill** tasks.
